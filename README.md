# File Secure Management System

A secure web-based application built using Flask that allows users to safely upload, lock, and manage files with authentication and encryption.

## Features
- User authentication (signup, login, logout)
- Secure file locking mechanism
- Encrypted file storage
- Password reset functionality
- Clean and responsive UI

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript

## Project Structure
- `app.py` – Main Flask application
- `cosmic_file_locker.py` – Core file locking & security logic
- `users.json` – User data storage
- `*.html` – Frontend pages
- `styles.css` – Styling
- `script.js` – Client-side logic

## How to Run Locally
1. Clone the repository
2. Install required dependencies
3. Run `python app.py`
4. Open browser and visit `http://localhost:5000`
